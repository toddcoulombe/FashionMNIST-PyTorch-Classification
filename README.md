#  FashionMNIST Image Classification with PyTorch

## Overview

This project demonstrates image classification using PyTorch and the FashionMNIST dataset.

## Project Results

- Fully Connected Neural Network Accuracy: 81.5%
- Convolutional Neural Network (CNN) Accuracy: 92.0%
- Improvement: +10.5%

## Dataset

- 60,000 training images
- 10,000 test images
- 28x28 grayscale images
- 10 clothing categories

## Model 1: Fully Connected Neural Network

Results:
- Accuracy: 81.5%

## Model 2: Convolutional Neural Network (CNN)

Results:
- Accuracy: 92.0%

## Technologies Used

- PyTorch
- DataLoader
- Neural Networks
- CNNs
- ReLU
- MaxPooling
- Adam Optimizer

## Key Learning

The CNN significantly outperformed the fully connected neural network by automatically learning spatial image features through convolutional and pooling layers. This project demonstrated how deeper architectures can improve classification accuracy while reducing manual feature engineering.
